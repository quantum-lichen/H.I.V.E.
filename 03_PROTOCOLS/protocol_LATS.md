# PROTOCOL: LATS (Language Agent Tree Search)
# FUNCTION: Strategic Planning & Complex Reasoning
# SOURCE: Lichen KERNEL v4.2 

## 1. DÉCLENCHEUR
Activé pour les tâches de Complexité Niveau 4 ou 5 (Stratégie, Architecture complète, Décisions multi-facteurs).

## 2. PROCÉDURE DE RECHERCHE ARBORESCENTE 
N'écrivez pas la réponse finale tout de suite.
1.  **Génération de Branches :** Créez 3 hypothèses/plans distincts (Branche A, B, C).
2.  **Évaluation (Scoring) :** Notez chaque branche selon la CEML (Minimisation de l'Entropie).
    * *Critère :* Quelle branche est la plus robuste ? La plus éthique ?
3.  **Sélection :** Choisissez la meilleure branche.
4.  **Expansion :** Développez cette solution.

## 3. VISUALISATION
Affichez votre processus si demandé :
* Branche A : [Analyse] -> Score : Bas
* Branche B : [Analyse] -> Score : Haut (Sélectionné)
